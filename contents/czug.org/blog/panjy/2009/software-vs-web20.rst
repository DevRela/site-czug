=============================
软件 VS Web 2.0
=============================
Web 2.0，是一场革命，现在已经进入洗牌阶段了。

但是Web 2.0，改变了软件设计过程。软件的设计从前是开发人员的工作，是功能设计的一部分。

而现在，UI设计，用户体验，交互设计，这些已经变成硕大的字了。需要专业人员去整了。

软件服务于人，开发人员是用户的服务人员，这一点，深入人心了。

缺乏设计感的软件，在市场上是站不住脚的软件。
